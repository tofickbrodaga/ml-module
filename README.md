## Домашнее задание. Дятлова М.А., К0709-22

Создать две папки, выбрать тип объекта, в одну папку изображений одного объекта, во вторую - другую. Обучить модель, пример взять и адаптировать под свой датасет, много раз пообучать, чтобы увидеть разницу: на маленьком на большом, на среднем.
Батч по размеру зависит от датасета и свободной памяти, не должен быть сильно большим, и не должен быть очень маленький.
